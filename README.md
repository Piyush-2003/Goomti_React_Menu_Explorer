# Goomti Menu Explorer — React App

Interactive menu engineering analysis tool built with React, 
using real dine-in sales data from Goomti Restaurant.

## 🔗 Live Demo
[View Menu Explorer](https://goomti-react-menu-explorer-wx9w.vercel.app/)

## Features
- **Search** — filter menu items in real time as you type
- **Category filter** — dropdown to filter by menu category
- **Sort options** — sort by revenue, volume or average price
- **Quadrant filter** — click Star/Puzzle/Plowhorse/Dog to filter by classification
- **Revenue bar** — visual bar showing each item's share of total revenue
- **Hover effects** — cards lift on hover for a polished UI feel

## Menu Engineering Quadrants
Every item is automatically classified using the Boston Matrix:
- ⭐ **Star** — high revenue + high volume → promote these
- ❓ **Puzzle** — high revenue + low volume → market more
